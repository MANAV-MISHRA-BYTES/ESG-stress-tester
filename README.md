# ESG Stress Tester

A Streamlit application that simulates the impact of Environmental, Social, and Governance (ESG) shocks on a stock portfolio.

## Features
- Fetches real-time stock prices via yfinance.
- Generates synthetic ESG scores for demonstration.
- Simulates 3 types of stress scenarios:
  1. Climate Policy (E)
  2. Social Unrest (S)
  3. Governance Scandal (G)
- Visualizes Portfolio Value at Risk (VaR).
